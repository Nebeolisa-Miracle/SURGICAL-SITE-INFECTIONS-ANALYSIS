## SURGICAL-SITE-INFECTIONS-ANALYSIS

### PROJECT OVERVIEW

Surgical Site Infections (SSIs) remain one of the major challenges facing healthcare systems worldwide, contributing to prolonged hospital stays, increased healthcare costs, and poorer patient outcomes.

This project analyzes surgical site infection data collected from California hospitals between 2021 and 2023 to uncover infection trends, evaluate hospital performance, and identify key factors associated with infection rates.

The analysis was carried out using Excel, Power BI and other tools, to transform raw healthcare data into actionable insights that can support evidence-based decision-making and improve patient care.


### PROJECT OBJECTIVES

The objectives of this analysis are to:

- Examine trends in Surgical Site Infections (SSIs) from 2021 to 2023.
- Evaluate hospital performance using the Standardized Infection Ratio (SIR).
- Identify hospitals and procedures with higher infection rates.
- Compare observed infections with predicted infections.
- Determine the percentage of facilities performing above acceptable infection thresholds.
- Provide recommendations to improve patient safety and healthcare outcomes.


### TOOLS USED

- Microsoft Excel - Data Cleaning
- SQL - Quering and EDA
- Power BI - Data Visualisations
- DAX - Calculations of Measures
  


###  DATASET DESCRIPTION

The dataset contains records of surgical site infections reported by hospitals across California between 2021 and 2023.


### KEY VARIABLES

- Reporting Year
- Hospital Name
- Procedure Type
- Number of Procedures
- Predicted Infections
- Reported Infections
- Excess Infections
- Standardized Infection Ratio (SIR)
- Number of Facilities


### KEY PERFORMANCE INDICATORS (KPIs)

###  KEY INSIGHTS

1. The Standardized Infection Ratio (SIR) showed a downward trend from 2021 to 2023, indicating improvements in infection prevention measures across healthcare facilities.
2. Although infection rates improved, the total number of reported infections increased over time, suggesting increased surgical procedures or better reporting systems.
3. A significant proportion of hospitals recorded SIR values greater than 1, indicating infection rates above expected benchmarks.
4. The analysis revealed noticeable differences between predicted infections and actual infections, highlighting areas requiring further investigation.
5. Excess infections were concentrated in a few hospitals, suggesting that infection-control challenges are more severe in specific facilities.
6. Some hospitals consistently performed below expected benchmarks, indicating differences in the effectiveness of infection-prevention measures.


###  RECOMMENDATIONS
1. Strengthen infection prevention protocols by enforcing proper sterilization procedures, maintaining operating room hygiene, and ensuring compliance with infection-control guidelines.
2. Increase monitoring of hospitals with high SIR values through regular audits and performance reviews to quickly identify and address problem areas.
3. Promote data-driven decision-making by using real-time dashboards to track infection rates and support timely interventions.
4. Provide regular staff training on hand hygiene, wound care, and other infection-control practices to reduce preventable infections.
5. Improve surveillance systems by implementing automated reporting tools to detect infection patterns and enable early intervention.
6. Support underperforming hospitals with additional resources and encourage the adoption of best practices from high-performing facilities.
